### МЕТОД ГАУССА
Программа решает систему линейных однородных уравнений с комплексными коэффициентами (действительная и мнимая части - рациональные числа)
Числа матрицы СЛУ считываются из файла, каждая строчка - одно комплексное число, представленное четырьмя целыми
Для сборки используйте cmakefile.txt
